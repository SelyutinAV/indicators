# Быстрый старт

## Первый запуск

1. Активируйте виртуальное окружение:
```bash
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

2. Создайте суперпользователя (если еще не создан):
```bash
python manage.py createsuperuser
```

3. Запустите сервер:
```bash
python manage.py runserver
```

4. Откройте http://127.0.0.1:8000/admin/ в браузере

## Первые шаги

1. **Создайте единицы измерения:**
   - Перейдите в "Единицы измерения"
   - Создайте несколько единиц (например: "Рубли", "Штуки", "Проценты")

2. **Создайте атомарные показатели:**
   - Перейдите в "Показатели"
   - Создайте показатель с типом "Атомарный"
   - Укажите min_value и max_value (например: 10 и 100)
   - Сохраните

3. **Сгенерируйте тестовые данные:**
   - Откройте карточку показателя
   - В разделе "Генерация тестовых данных" укажите:
     - Начальная дата (например, месяц назад)
     - Конечная дата (сегодня)
     - Отметьте "Сгенерировать тестовые данные"
   - Сохраните

4. **Создайте агрегатный показатель:**
   - Создайте новый показатель с типом "Агрегатный"
   - В поле "Формула" укажите, например: `[Показатель1] + [Показатель2]`
   - Сохраните

## Примеры формул

- Сложение: `[Показатель1] + [Показатель2]`
- Деление: `[Показатель1] / [Показатель2]`
- Процент: `[Показатель1] * 100 / [Показатель2]`
- Среднее: `([Показатель1] + [Показатель2]) / 2`
- Смешанные: `[Агрегатный показатель] * [Атомарный показатель]`
